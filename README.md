orange-privacy-payment
======
Prototype implementing the random-value token payment solution. 

Requirements:
----
* truffle 
* ganache-cli
* python version>3 

Set up the environment:
----
```pip3 install requirements.txt```

Launch the platform: 
----
* (1) ganache
```./scripts/launch.sh```

* (2) scenario:
```cd back && python3 test_scenario.py```
